
  # Civic Infrastructure Management App

  This is a code bundle for Civic Infrastructure Management App. The original project is available at https://www.figma.com/design/DG8GYm4scKcJf8RJ6LtOWY/Civic-Infrastructure-Management-App.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm --prefix backend i` to install backend dependencies.

  Run `npm run dev` to start the development server.

  Run `npm run dev:backend` in another terminal to start the REST API backend on `http://localhost:8000`.
